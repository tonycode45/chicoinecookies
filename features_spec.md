# Features Specification – Chicoine Cookies

## Essential Features
- User registration, login, logout
- Product browsing + filtering
- Product details page
- Add to cart (drawer + dedicated page)
- Checkout flow (UI only)
- Subscriptions page + plan comparison
- Email list sign-up
- Story page
- Contact page
- FAQ

## Elite Features
### Build-Your-Own Box
- Drag & drop cookie selector
- Visual "box filling up" interface
- Calorie/protein per cookie display (optional)

### Cookie Lab – Limited Drops
- Countdown timers
- Seasonal + experimental flavours
- “Drop ends soon” banners

### Loyalty / Rewards System
- Points tracker
- Level badges (Bronze → Silver → Gold)
- Free box progress bar UI

### Gifting Flow
- Gift message
- Custom card design selection
- Scheduled delivery UI

### Corporate / Event Orders
- Form with budget, headcount, packaging needs

### Cookie Personality Quiz
- Multi-step quiz with illustrations
- Personalized flavour recommendations

### UGC & Reviews Wall
- Customer photos (UI only)
- Star ratings
- “Top Cookie Fan” badge

### Behind the Scenes
- Video embeds
- “How we bake” sections

### Nutrition & Allergens
- Filters: nut-free, gluten-friendly, dairy-free
- Ingredient tables

## Dashboard Features
- Orders
- Subscription management
- Loyalty progress
- Profile settings
